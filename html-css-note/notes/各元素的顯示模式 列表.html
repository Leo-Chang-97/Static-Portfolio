HTML 元素顯示模式

📌 塊級元素（Block Elements）

特性： 塊級元素會獨佔一整行，並且會自動擴展寬度以填滿父容器。它們可以包含其他塊級或行內元素。

根元素：

<html>, <body>

排版相關標籤：

<h1> - <h6>, <hr>, <p>, <pre>, <div>

列表相關標籤：

<ul>, <ol>, <li>, <dl>, <dt>, <dd>

表格相關標籤：

<table>, <tbody>, <thead>, <tfoot>, <tr>, <caption>

表單相關標籤：

<form>

📌 行內元素（Inline Elements）

特性： 行內元素不會換行，只佔據它所需的空間，寬度與內容匹配。行內元素通常只能包含文本或其他行內元素。

文字格式化標籤：

<br>, <em>, <strong>, <sup>, <sub>, <del>, <ins>

超連結與標籤相關：

<a>, <label>, <span>

📌 行內區塊元素（Inline-block Elements）

特性： 行內塊級元素與行內元素類似，但可以設定寬度和高度，而且不會強制換行。

圖片相關標籤：

<img>

表格單元格：

<td>, <th>

表單相關標籤：

<input>, <textarea>, <select>, <button>

內嵌框架：

<iframe>
