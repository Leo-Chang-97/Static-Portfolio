CSS 優先級權重分級

類型 權重 範例
行內樣式 (Inline Styles) 1,0,0,0 <div style="color: red;">
ID 選擇器 (ID Selectors) 0,1,0,0 #my-id
類別選擇器、屬性選擇器、偽類 (Class, Attribute, Pseudo-class) 0,0,1,0 .my-class, [type="text"], :hover
元素選擇器、偽元素 (Element, Pseudo-element) 0,0,0,1 div, p::before
通用選擇器、組合符 (Universal Selector, Combinators) 0,0,0,0 *, >, +, ~
繼承 (Inherited Styles) n/a color, font-size
默認 (User Agent Default Styles) n/a button, input
